폴더명 : v00_instell_anaconda
파일명 : v00_install anaconda.md

1.아니콘다 설치 : https://repo.anaconda.com/archive/ 로 들어가 구저번 설치 이때 설치 경로를 복사해 둔다.
2.아나콘다 설치 완료 후 환경 변수 설정 : 윈도우 검색 창에 시스템 환경 변수 설정을 들어간후 환경 변수로 들어가 시스템 변수에서 path를 찾은후 더블 클릭으로 들어간다.
3.그후 새로 만들기에 들어가 아나콘다의 복사해놨던 설치 경로를 넣어주고 또한 새로 만들어 해당 경로 뒤에 \Library\bin과 \Library,\Scripts를 각각 만들어 총 4개를 만들어주면 환경변수 설정과 설치가 끝난다.
4.그후 cmd를 실행후 conda를 입력해서 변수가 제대로 작동하는지 확인후 다시 한번 더 conda create -n py39 python=3.9를 입력한다.
conda create -n은 새로운 가상환경을 만드는 것이고 그 뒤의 py39가 이름 그 뒤는 알아보기 쉽게 사용할 프로그렘(파이선)의 버전을 적어놓은것이다.
5.이렇게 되면 아나콘다의 설치, 아나콘다의 환경변수 설정, 새로운 가상환경의 작성이 완료된다.
